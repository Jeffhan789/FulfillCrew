# Security Policy

Security fixes target the latest release and the `main` branch.

Report vulnerabilities privately through GitHub's **Report a vulnerability** flow in the Security tab. Include affected versions, reproduction steps, impact, and mitigation ideas. An acknowledgement is expected within 7 days.

Do not include real customer, payment, warehouse, or credential data. FulfillCrew is an educational system and should not be used as a production order or fraud-decision platform without an independent security, privacy, and reliability review.
