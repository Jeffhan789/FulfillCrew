# Course Mapping

## COMP315 Cloud Computing

- JavaScript data cleaning from CA1 becomes the product data pipeline.
- React e-commerce frontend from CA2 becomes the product and checkout interface.
- FastAPI provides product, order and agent APIs.
- Docker files prepare the project for Linux and cloud deployment.

## COMP310 Multi-Agent Systems

- Order Agent receives checkout requests.
- Inventory Agent checks stock and reserves inventory.
- Coordinator Agent runs a simplified Contract Net Protocol.
- Warehouse Agents submit bids using workload, distance, stock and processing speed.
- Demand Prediction Agent and Fraud Detection Agent connect intelligent services into the order workflow.

## ELEC320 Neural Networks

- Demand Prediction module maps product features to predicted next 7-day sales.
- Fraud Detection module maps order features to a 0-1 risk score.
- Product Category Classifier demonstrates labelled classification using product fields.
